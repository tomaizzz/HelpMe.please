# HelpMe.please

Sistema simples de gerenciamento de tarefas desenvolvido como MVP para a disciplina de Fábrica de Software.

## Descrição

O projeto tem como objetivo ajudar o usuário a organizar suas tarefas diárias de forma simples e prática, evitando desorganização e esquecimento de atividades.

## Funcionalidades

- Cadastro de tarefas
- Listagem de tarefas
- Marcação de tarefas como concluídas
- Atualização dinâmica da interface
- Validação de entrada (não permite tarefas vazias)

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Estrutura do projeto

- index.html → estrutura da interface
- style.css → estilização da aplicação
- script.js → lógica e funcionamento do sistema

## Observações

Este projeto representa um MVP (Produto Mínimo Viável), portanto não inclui funcionalidades como autenticação de usuários, banco de dados ou integrações externas.
